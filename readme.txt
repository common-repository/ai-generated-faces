This extension grants access to the generated faces API which uses AI to generate random well designed human faces.

== Installation ==

To use it :

- Make sure to have Elementor page builder activated
- Extract files and put them in the extension wordpress folder
- Activate
- go to https://generated.photos/ website to create a free account
- Copy your API key in your profile page
- On elementor, drag the AI_Faces widget onto your page and select options (don't forget to put in you API key for it to work).

- Enjoy :=)


== Frequently Asked Questions ==


= What is it about ? =

It's all about AI generated faces from generated faces website.

= Why are some combinations not working ? =

Some combinations like eyes colors, hair length, ... generate no result. Simply try another combination.
